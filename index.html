Everything Agents
Need to Remember.

Hybrid search, knowledge graph, contradiction detection — all from a single write.

⬡
Hybrid Search
pgvector semantic similarity + full-text keyword matching + knowledge graph expansion. Ranked by similarity, importance, freshness.
◎
Auto-Enrichment
Single-pass LLM enrichment on every write. Auto-classifies into 14 memory types, generates title, summary, tags, importance score.
∿
Karpathy Loop
Agents report success/failure after acting on recalled memories. System reinforces what works, generates preventive rules on failure.
⚡
Tenant Isolation
Row-level database separation per tenant. PII auto-detected and quarantined. Every memory stamped with visibility scope at write time.
✓
MCP Server
9 tools via Model Context Protocol. Connect Claude Desktop, Claude Code, Cursor, Windsurf, or OpenClaw with a URL and API key.
⊕
Knowledge Graph
People, orgs, concepts extracted into entities and relations on every write. Semantic entity resolution auto-merges duplicates at 0.85 cosine.
